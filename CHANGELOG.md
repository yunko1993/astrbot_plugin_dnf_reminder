# Changelog

## 1.6.2 - 2026-06-16

- 将插件配置改为分组展示：发送设置、群目标设置、@ 提醒设置。
- 群目标配置改为列表控件，便于在 AstrBot 控制台添加多个群。
- 兼容读取旧版本的平铺配置，升级后无需手动迁移配置。
- 修正 `metadata.yaml` 和 README 中的 GitHub 仓库地址。

## 1.6.1 - 2026-06-16

- 添加提醒时支持直接识别群聊中的真实 `@成员`。
- 保留手写 `@QQ号` 的兼容写法。
- 保存提醒内容时会清理开头的 CQ at 标记或 `@QQ号` 前缀。

## 1.6.0 - 2026-06-16

- 添加提醒时支持可选 `@QQ号`。
- 群消息发送时优先生成真实 At 组件。
- 提醒列表展示已配置的 @ 目标。

## 1.5.1 - 2026-03-30

- 优化定时任务调度，避免 scheduler 尚未就绪导致提醒未注册。
- 定时触发时将发送任务调度回主事件循环。
- 增加旧数据目录迁移和兼容读取。

## 1.5.0 - 2026-03-30

- 支持发送到配置群。
- 支持群消息 @ 所有人。
- 新增插件配置 schema。
