# DNF 私人提醒秘书 (AstrBot Plugin)

给 DNF 玩家准备的每日定时提醒插件。你可以在群聊或私聊里添加提醒，机器人会在指定时间通过主动消息发给你。

## 安装

在 AstrBot 控制台执行：

```text
/plugin install https://github.com/yunko1993/astrbot_plugin_dnf_reminder
```

或者手动把仓库放到插件目录中。

## 指令

所有指令前缀均为 `/提醒`。

| 功能 | 指令示例 |
| :--- | :--- |
| 添加提醒 | `/提醒添加 10:00 领取心悦增幅器` |
| 添加并 @ 指定人 | `/提醒添加 10:00 @123456789 领取心悦增幅器` |
| 查看列表 | `/提醒列表` |
| 删除提醒 | `/提醒删除 0` |
| 立即测试 | `/提醒测试` |

## 数据文件

提醒数据保存在 `plugin_data/dnf_personal_reminder/reminders.json`。

为兼容旧版本，插件启动时也会尝试读取旧目录中的 `reminders.json`。

## 说明

- 插件会优先复用历史数据目录，避免更新后读不到旧配置。
- 新增提醒后会立即重新注册定时任务。
- 定时触发时会把消息发送调度回主事件循环，避免线程里直接发消息失败。
- 可以在插件配置里开启“发送到配置群”，并填写群目标列表，让提醒同步发到群里。
- 控制台配置页支持直接维护“提醒列表”，每条提醒都可以配置标题、时间、内容、发送目标和 @ 对象；@ 指定人支持填写多个 QQ 号。
- 控制台提醒的发送目标可以填完整会话 ID，也可以填纯群号；纯群号会按“纯群号平台前缀”自动补成群会话 ID。NapCat 常见前缀类似 `napcat_1`，也可以填 `auto` 让 `/提醒测试` 从当前会话临时学习。
- 添加提醒时既支持手写 `@QQ号`，也支持在群里直接 `@成员`，发送到群时会优先真正 `@` 这个人。
- 开启“群消息时 @所有人”后，会优先尝试发送真正的 `@all` 组件；是否生效取决于 AstrBot 适配器和机器人权限。
